
# 舒适系统

### 关闭大灯清洗

``` yaml title="登录密码: 31347"
单元 09 → 编码:
head_lamp_wash: not_installed
```

### 舒适折叠后视镜

``` yaml
单元 46 舒适系统中央模块 → 编码:
mirror_retraction_at_comfort_close: 激活
```
之后需在「车机」菜单中取消勾选「落锁时折叠」

### 驾驶员舒适入座模式

``` yaml
单元 36 驾驶员座椅调节单元 → 编码:
字节 3 – 位 6: 激活 
→ 应用（需重启单元）
```
座椅向后移动以便更舒适地入座。  
需在车机中激活「入座辅助」和「将设置保存到无线钥匙」

### 乘客舒适入座模式

``` yaml
单元 06 乘客座椅调节单元 → 编码:
字节 3 – 位 6: 激活 
→ 应用（需重启单元）
```

与驾驶员座椅不同，乘客座椅在发动机运行时也工作，只需开门、关门即可

### 用驾驶员侧按键调节乘客座椅。座椅对称。

``` yaml
单元 36 驾驶员座椅调节单元 → 编码:
字节 2 – 位 5: 激活 
字节 6 – 位 1: 激活 
→ 应用（需重启单元）
```
``` yaml
单元 06 乘客座椅调节单元 → 编码:
字节 2 – 位 5: 激活 
字节 6 – 位 1: 激活 
→ 应用（需重启单元）
```

车机中会出现相应菜单。
