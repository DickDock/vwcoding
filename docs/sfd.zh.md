# SFD 与 SFD2 — VAG 诊断保护

!!! warning "重要"
    从 **MQB Evo 平台（Golf 8、Octavia 4、Leon 4、ID.3 等）** 开始，许多写入操作（编码、适配、加载参数集）都需要解锁 **SFD**（*Schutz Fahrzeug Diagnose*，车辆诊断保护）。

    在**约 2024 车型年起**的车辆上，还额外引入了 **SFD2** —— 每一次写入操作都需要单独的令牌。

## SFD（第一版，约 2020 年起）

|                |                                                                              |
|----------------|------------------------------------------------------------------------------|
| **出现范围**   | MQB Evo、MEB、部分新款 MLB                                                    |
| **保护内容**   | 对控制单元的写入（编码、适配、参数集）                                        |
| **如何解锁**   | ODIS Service **在线** + GeKo / UMA 账号（经销商或合作伙伴权限）              |
| **有效期**     | 通常 **90 分钟** 或直到会话结束                                              |
| **工具**       | ODIS Service、ODIS Engineering；OBD11 —— 需支持 SFD 且账号有效              |

!!! tip ""
    SFD 车辆的特征：网关编号形如 **5WA 907 530** 及更新，以及本站 MQB-Evo 页面上的 `:octicons-verified-24: SFD: yes` 标记。

### ODIS 中的典型操作顺序

1. 连接 VAS6154 / ENET，启动 ODIS Service online。
2. 读取控制单元 → **Access authorization**（访问授权）→ 在线申请 SFD 令牌。
3. 解锁成功后再执行编码或适配。
4. 会话结束后控制单元重新受保护。

## SFD2（约 2024 车型年起）

SFD2 是为满足 **UNECE R155/R156**（网络安全）要求而引入的。

| SFD v1                                          | SFD v2                                              |
|-------------------------------------------------|-----------------------------------------------------|
| 访问诊断会话 / 基础解锁                          | 保护**具体某一次写入**（编码、适配、参数集）        |
| 对一个控制单元的一次操作窗口                      | **每次写入操作**都需单独的校验和令牌                |

!!! danger ""
    在许多 2024+ 车辆上，**VCDS 与部分第三方 dongle 无法获取 SFD2 令牌**。写入请使用带有效 VW Group 订阅的 **ODIS Service online**。

### 含 SFD2 的车型示例（不完全列表）

VW Golf 8（2024.04 起）、Polo AE1、T-Roc、Tiguan Allspace、Touran、Taigo、ID.3/ID.4/ID.5（2024+）、Skoda Octavia NX (GP1)、Fabia PJ、Scala、Kamiq、Karoq、Seat Leon KL/KU、Cupra Formentor 等 —— 请按生产日期与 PR 号确认。

## 工具兼容性

下表仅供参考 —— 具体能力取决于软件版本、订阅以及车辆的**生产日期**。

| 工具                        | SFD v1 (Evo ~2020–2023) | SFD2 (2024+ MY) | 备注                                                          |
|-----------------------------|-------------------------|-----------------|---------------------------------------------------------------|
| **ODIS Service online**     | ✅                       | ✅               | 标杆；需 GeKo/UMA 与 VAS6154/ENET                            |
| **ODIS Engineering**        | ✅                       | ✅               | 参数集、刷写；对账号要求相同                                  |
| **OBDeleven Pro** (SFD)     | ✅ 经常                  | ⚠️ 受限         | 取决于车型支持与 OBD11 服务器                                 |
| **VCDS**                    | ⚠️ 少数                 | ❌ 通常不支持   | 读取常可用；对 SFD2 的写入不支持                              |
| **Car Scanner / VAG OBD**   | ⚠️                      | ❌               | 主要是实时数据；不针对受保护单元的写入                        |
| **VCP、其他 dongle**        | ⚠️                      | ❌               | 请核对最新的 SFD2 支持列表                                    |

!!! tip ""
    在购买"适用于 Golf 8 / Octavia 4 2024"的工具前，请向卖家确认其对**你的 VIN 的 SFD2** 支持，而不仅是 SFD v1。

## 哪些控制单元最常被保护

在 **MQB-Evo / MEB** 上，以下单元的写入通常经由 SFD：

| 单元        | 用途                                    | SFD    | SFD2 (2024+) |
|-------------|-----------------------------------------|--------|--------------|
| **19**      | 网关、拓扑                              | 是     | 是           |
| **09**      | BCM、灯光、舒适                         | 经常   | 经常         |
| **17**      | 仪表盘                                  | 经常   | 经常         |
| **5F**      | 信息娱乐 / ICAS                         | 是     | 是           |
| **A5**      | 辅助摄像头                              | 是     | 是           |
| **44**      | 转向助力（Travel Assist、ALC）         | 是     | 是           |
| **13**      | ACC / Front Assist                      | 有时   | 有时         |
| **76**      | Park Assist                             | 有时   | 有时         |
| **4B**      | IQ.Light / MFK                          | 是     | 是           |
| **C5 / 8C** | 高压电池（MEB）                        | 是     | 是           |

MQB-Evo 页面上的 `:octicons-verified-24: SFD: yes/no` 标记针对的是**具体某条编码**，而非整个控制单元。

## 常见错误及处理

| 症状                                         | 可能原因                                | 处理                                                  |
|----------------------------------------------|-----------------------------------------|-------------------------------------------------------|
| "Access denied" / 写入被拒                    | 未解锁 SFD 或令牌已过期                  | ODIS online → Access authorization → 申请新 SFD       |
| 2024+ 上 SFD 有效但写入仍被拒                  | 需要针对该操作的 **SFD2** 令牌          | ODIS Service online；VCDS/OBD11 通常无能为力           |
| SFD 正常，但通道不生效                         | 单元 SW 不对或不存在该通道              | 核对 SW 版本；并非所有 PR 都有全部通道                 |
| 需要打开机盖 / 拉手刹                          | MQB 2021+ 的 ODIS 条件                  | 写入前先满足（见 [MQB / index](MQB/index.md)）        |
| 刷写参数集后报错                               | 参数集不兼容                            | 备份；加载适配你 HW/SW 的参数集                        |
| 网关 19 —— 拓扑错误                            | 单元安装不正确                          | 不要"盲"编码；核对 PR 与 coding 19                     |

## MQB（Evo 之前）

在经典 **MQB**（Octavia A7、Tiguan AD1、Evo 改款前的 Passat B8）上**通常没有 SFD**。只需控制单元的 login 码即可。

对于 **2021 车型年及更早** 的 MQB，向某些单元写入时，ODIS 可能要求：

- 打开机盖；
- 拉起手刹。

参见 [MQB / 通用信息](MQB/index.md)。

## 建议

1. 更改前**先备份**控制单元的编码与适配。
2. 注意 MQB-Evo 页面上的 **SFD: yes/no** 标记。
3. 对于新车，请预留 **ODIS 在线会话**时间。
4. 不要在未核实 SFD 的情况下，沿用仅基于 VCDS 的旧教程去处理 Evo 与 MEB。
