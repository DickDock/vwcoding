# 其他功能

### 转速表和车速表指针自检
点火开启时，转速表和车速表指针从最小位置走到最大位置。没有任何实用功能，但看起来很炫！
``` yaml
单元 17 → 适配: 
通道「IDE01087」→ 测试 → 选择「active」→ 执行
```

### 更改报警器警笛声音
可以让原厂防盗警笛的声音更具个性。

!!! note "4 种声音"
    0 – 调制十次，  
    1 – 断续十次，  
    2 – 调制单次，  
    3 – 断续单次。
  
``` yaml
单元 09 → 适配: 
通道 13 → 测试 → 输入所需值 → 保存
```

!!! warning "注意"
    调制声 – 听起来像「呜 – 呜 - 呜」，断续声 – 像「嘀 – 嘀 – 嘀」。
  
### 通过门锁解除原厂防盗 
如果此功能激活，则可通过门锁解除车辆的原厂防盗，即此过程不涉及无线电子，仅靠钥匙金属片即可。  
也许对某些人有用，但还是最好关闭此功能，否则不法分子可通过普通机械破坏开门。
``` yaml
单元 09 → 编码 → 长编码:
字节 6 – 位 2: 停用
```

### 更改保养间隔
出于各种原因，可能需要更改保养间隔的距离和时间。
``` yaml
单元 17 → 适配: 
通道「IDE01153」→ 输入所需的公里值 → 保存
通道「IDE001154」→ 输入所需的天数值 → 保存
```

### 重置保养间隔
没有 MaxiDot 的人可通过 VAG-COM 重置保养间隔 – 相应地把距下次保养的公里数和天数清零。
``` yaml
单元 17 → 适配: 
通道「IDE00510」→ 输入「0」→ 保存
通道「IDE00511」→ 输入「0」→ 保存
```

!!! warning "注意"
    也可以用程序主菜单中的「重置」按钮来重置，但那样就没那么有情调了！
  
### 关闭安全气囊
关闭某个安全气囊的需求可能出于各种情况，例如重新缝制内饰、实际无气囊、宗教原因等。  
无论如何，每个气囊都可以单独关闭，相应地，该气囊的故障码也会消失。  
没有充分理由就关闭安全气囊前，请三思！

!!! note "说明"
    通道 01 – 右侧（乘客）正面气囊 – 在杂物箱上方；  
    通道 02 – 左侧（驾驶员）正面气囊 – 在方向盘内；  
    通道 03 – 右侧（乘客）侧气囊 – 在座椅靠背侧面；  
    通道 04 – 左侧（驾驶员）侧气囊 – 在座椅靠背侧面；  
    通道 07 – 右侧（乘客）「气帘」– 在前柱内；  
    通道 08 – 左侧（驾驶员）「气帘」– 在前柱内。  
  
``` yaml
单元 15 → 适配: 
通道号 → 输入「1」→ 保存
```

!!! warning "注意"
    我坚决反对为掩盖车上气囊缺失、以欺骗新买家为目的而关闭安全气囊。  
    如果出于此目的关闭气囊，那只能听天由命，而我会朝你脸上啐一口！
