# 灯光

### 开启 DRL
以多种方式激活 DRL（日间行车灯）。

!!! note "说明"
    有 4 种方式：
    A – 通过前雾灯内的附加段实现 DRL（即 DRL 本身）；  
    B – 通过示廓灯实现 DRL；  
    C – 通过近光实现 DRL；  
    D – 通过前雾灯实现 DRL（有 2 种取值：64 – 开启，E4 – 开启, 可通过 MaxiDot 关闭）。
``` yaml
单元 09 → 编码 → 长编码:
A) 字节 15 – 位 6: 激活 
   字节 15 – 第 7 位: 激活
B) 字节 24 – 位 7: 激活
C) 字节 11 – 位 2: 激活
D) 字节 14 – 位 1: 激活
   字节 15 – 位 7: 激活
```

!!! warning "注意"
    如果没有 MaxiDot，则方式 A 和 D 中第 15 字节的第 7 位可以不开启。

### DRL 在「O」位的工作
灯光开关在「Auto」位时，会根据光照在 DRL 和近光之间切换。开启此功能后，DRL 仅在「Auto」位工作；关闭功能后，DRL 在「O」位时也会点亮。
``` yaml
单元 09 → 编码 → 长编码:
字节 26 – 位 3: 停用（或激活）
```

### 通过 DRL 实现示廓灯
即使把开关切到「示廓灯」或「近光」模式，DRL 仍继续点亮。可帮助那些用「睫毛」贴住前示廓灯的人。
``` yaml
单元 09 → 编码 → 长编码:
字节 25 – 位 1: 激活
```

### 拉手刹时关闭 DRL
车辆挂着手刹停着时 DRL 关闭，一旦松开「手刹」，DRL 点亮。
``` yaml
单元 09 → 编码 → 长编码:
字节 11 – 位 0: 激活
```

### 开双闪时关闭 DRL
车辆开着「双闪」时，DRL 关闭。
``` yaml
单元 09 → 编码 → 长编码:
字节 15 – 位 3: 激活
```

### 通过 DRL 实现频闪效果
开启远光时临时关闭 DRL。如果「晃」远光，就会得到不错的频闪效果。
``` yaml
单元 09 → 编码 → 长编码:
字节 12 – 位 1: 激活
```

!!! warning "注意"
    如果用近光作为 DRL 接入，就会得到「超级频闪」（氙气车不推荐）。

### 通过前雾灯实现频闪效果
开启远光时临时关闭前雾灯。如果「晃」远光，就会得到不错的频闪效果。
``` yaml
单元 09 → 编码 → 长编码:
字节 14 – 位 2: 激活
```

### 开后备厢门时的车内照明
默认情况下，开后备厢门时车内会亮灯（前提是车内照明开关置于「开门亮灯」位置）。  
不需要的人可以关闭。
``` yaml
单元 09 → 编码 → 长编码:
字节 20 – 位 1: 停用
```

### Corner 功能
Corner 功能用前雾灯照亮转弯。仅在 40 km/h 以下且与近光配合时工作。

!!! note "说明"
    有 2 种方式：  
    A – 根据方向盘传感器位置开启；  
    B – 根据方向盘传感器位置和转向灯开启状态共同开启。
  
``` yaml
单元 09 → 编码 → 长编码:
字节 14 – 位 0: 激活 → 第 7 位: 激活
A) 字节 17 – 位 0: 停用
B) 字节 17 – 位 0: 激活
```

!!! warning "注意"
    如果关闭第 14 字节的第 0 位，则不仅 Corner 失效，前雾灯也会失效。
  
### 倒车时关闭 Corner
Corner 功能默认在倒车时也工作。如果不需要，可以关闭。
``` yaml
单元 09 → 编码 → 长编码:
字节 26 – 位 6: 停用
```

### Coming Home 功能
Coming Home 功能在锁车后让近光（或前雾灯）和示廓灯保持点亮一段时间，仿佛为你照亮回家的路。

!!! note "2 种工作方式"
    A – 通过近光；  
    B – 通过前雾灯。  
!!! note "2 种触发方式"
    C – 开驾驶员车门后；  
    D – 熄火后。  
!!! note "2 种控制方式"
    E – 自动（由光照传感器激活）；  
    F – 手动（熄火后需「晃」一下远光）。
  
``` yaml
单元 09 → 编码 → 长编码:
A) 字节 17 – 位 5: 激活
B) 字节 17 – 位 3: 激活
C) 字节 12 – 位 0: 激活
D) 字节 12 – 位 0: 停用
E) 字节 12 – 位 2: 停用
F) 字节 12 – 位 2: 激活
```
``` yaml
单元 09 → 适配: 
通道 23「Coming Home」→ 测试 → 输入工作时长值（秒，10 到 60） 
→ 保存
```

### Leaving Home 功能
Leaving Home 功能在用原厂遥控器开锁后让近光和示廓灯点亮一段时间，仿佛为你照亮走向车辆的路。在夜间于拥挤的停车场找车时也很有用。
``` yaml
单元 09 → 编码 → 长编码:
字节 17 – 位 6: 激活
```
``` yaml
单元 09 → 适配: 
通道 24「Leaving Home」→ 测试 → 输入工作时长值（秒，10 到 60） 
→ 保存
```

### 舒适转向灯
舒适转向灯（即高速公路模式）在轻触转向灯拨杆（不把拨杆固定在「开启」位置）时，让转向灯闪烁一定次数。

!!! note "说明"
    在 087 J 和 087 Q 单元上，只有把 MaxiDot 复位到出厂设置后才能更改。
  
``` yaml
单元 09 → 适配: 
通道 31「转向灯闪烁（周期）」→ 测试 → 输入闪烁次数值（1 到 5）→ 保存
```

!!! warning "注意"
    默认值为「3」。

### 锁车的视觉确认
用原厂防盗开/关车辆时转向灯闪烁。

!!! note "说明"
    有 2 个值：0 – 关闭，1 – 开启。
  
``` yaml
单元 09 → 适配: 
通道 17 → 测试 → 输入所需值 → 保存
```

### 紧急双闪
在急刹（紧急）制动时触发，表现为刹车灯或转向灯（双闪）快速闪烁。

!!! note "2 种工作方式"
    A – 通过刹车灯；  
    B – 通过转向灯。  
  
``` yaml
单元 09 → 编码 → 长编码:
A) 字节 16 – 位 1: 激活
B) 字节 16 – 位 2: 激活
```

!!! warning "注意"
    同时激活两种方式时，紧急双闪将完全无法工作。
  
### Highway 模式
此功能在以超过 140 km/h 长时间行驶时，自动开启近光和示廓灯，即所谓的欧洲 Highway（或 Autobahn）模式。  
以低于 140 km/h 行驶超过 2 分钟时，该模式自动关闭。
``` yaml
单元 09 → 编码:
2 单元 RLS → 长编码:
字节 0 – 位 0: 激活
```

### 大雨时开启近光
前雨刮在自动模式下工作超过 15 秒、或被切到常开模式时，自动点亮近光。  
此功能默认开启。此编码将其关闭。
``` yaml
单元 09 → 编码:
2 单元 RLS → 长编码:
字节 0 – 位 1: 停用
```

### 调节大灯和尾灯亮度
``` yaml
单元 09 → 编码 → 长编码:
字节 19 – 位 0-3: 从列表中选择
字节 19 – 位 4-7: 从列表中选择
```

!!! warning "注意"
    第 0-3 位调节大灯亮度，第 4-7 位调节尾灯亮度

### 调节 DRL 亮度
所有 DRL 的亮度同时调节（如果有人额外把某些灯设为 DRL）。
``` yaml
单元 09 → 编码 → 长编码:
字节 24: 设定值（百分比）
```
!!! warning "注意"
    默认为 92%。
  
### 示廓灯模式下 LED 的亮度
在 Octavia A5FL RS 上，使 LED 灯在示廓灯模式下更亮。  
在 Octavia A5FL 上，此时 DRL 会与示廓灯一起点亮。  

!!! note "说明"
    如果已激活通过 DRL 的频闪效果，则会得到又一种「频闪」效果。
  
``` yaml
单元 09 → 编码 → 长编码:
字节 25 – 位 1: 激活
```

### 关闭牌照灯灯泡检测
如果后牌照灯灯罩里装了 LED 灯，就会报错。为消除报错，关闭灯泡检测。
``` yaml
单元 09 → 编码 → 长编码:
字节 17 – 位 7: 激活
```

### 关闭脚部照明灯泡检测
如果脚部照明灯罩里装了 LED 灯，就会报错。为消除报错，关闭灯泡检测。
``` yaml
单元 09 → 编码 → 长编码:
字节 25 – 位 3: 激活
```

## 近光「氙气」模式
安装氙气灯时，为消除非自愿的频闪效果，可开启此模式。但实践表明，这完全无法防止 BCM 单元中的通道烧毁 – 只是时间和镇流器质量问题。
``` yaml
单元 09 → 编码 → 长编码:
字节 14 – 位 4: 激活
```

## 关闭大灯灯泡正常性检测
在装有非原厂氙气、中国产大灯，或无附加负载的 LED 灯时需要。
``` yaml
单元 09 → 编码 → 长编码:

---近光检测
字节 14 – 位 3: 开启 (Bi-Xenon 大灯（带遮光片）- 已安装)
字节 14 – 位 4: 开启 (头灯检测不激活 (无解码器的 LED 和 Bi-Xenon))
---远光检测
字节 17 – 位 2: 开启 (无附加远光的 Bi-Xenon)
字节 25 – 位 2: 开启 通过遮光片的远光 – 激活/已安装
```
