# 多媒体

### 改善原厂车机音效
对预设均衡器进行更优的设定。音质接近带 12 扬声器的 Superb。
``` yaml
单元 56 → 编码 → 长编码:
字节 5 – 位 0-7 → 选择 04: Seat Altea XL (5P), Škoda Superb (3Т)
```

### 更改原厂车机的自主工作时长
钥匙拔出点火锁后，车机默认工作 30 分钟。这个时间可以延长。  
无需担心蓄电池状态 – 蓄电池负载监控会在电量临界时切断用电设备（包括车机）。  
自动关闭值以分钟为单位输入。  
要停用此功能，需输入值「255」。  
``` yaml
单元 56 → 适配: 
通道 3 → 输入所需值 → 执行
```

### 仪表盘和 Swing 原厂车机汉化
更改仪表盘语言时，也会更改 Swing 原厂车机 (RCD-310) 的语言，但仅限车机生产于 2011 年下半年及之后。
``` yaml
单元 17 → 适配: 
通道「IDE00343」→ 选择所需语言 → 保存
```

### 在 Swing 上显示车速
所谓的「驾校功能」可在车机屏幕上显示当前车速。读数仅在车机本身开启时显示。
``` yaml
单元 56 → 适配: 
通道 2 → 测试 → 输入值「1」→ 保存
```

!!! warning "注意"
    开启：发动机已启动、车机关闭 – 按下右「旋钮」并开机。 
    关闭：发动机已启动、车机关闭 – 按下「Eject」并开机。
  
### Swing / Bolero / Columbus 维修菜单
从车机维修菜单可获取大量信息并进行一些设定。
``` yaml
单元 56 → 适配: 
通道 50 → 测试 → 输入值「1」→ 保存
```

### 更改 Columbus 的 DVD 区域
不太清楚有何用，但原厂 Columbus 车机可以更改 DVD 区域。
``` yaml
单元 56 → 适配: 
通道 130 → 选择所需区域 → 保存
```
 
!!! warning "注意"
    值「0」开放所有区域。
  
### 更改 Columbus 配色方案
RNS-510 Columbus 标准配色为绿色。但可改为 VW 的灰红配色。

!!! note "说明"
    下拉列表中没有这个值，因此需要通过二进制码窗口来开启相应的位。  
    保存编码后，需同时按住（约 10 秒）「Eject」「麦克风」和「Setup」键来重启车机。
  
``` yaml
单元 56 → 编码 → 长编码:
字节 7 – 位 4: 激活
字节 7 – 位 5: 激活
```

### Bluetooth-Audio
带 Bluetooth 模块时，原厂车机默认允许通过 Bluetooth-Audio 播放音乐，即从手机播放。 
如果觉得此功能过于强制而恼人，可以关闭。

``` yaml
单元 77 → 编码 → 长编码:
字节 4 – 位 0: 激活
```

### 手机连接 Bluetooth 模块的提示音
默认情况下，手机连接/断开 Bluetooth 模块时，车机声音会临时关闭并发出提示音。  
关闭此功能可去掉连接提示音，且手机连接时不关闭车机声音。  
来电和通话时车机声音的自动关闭仍保留。  
``` yaml
单元 77 → 适配: 
通道「IDE02504」→ 设为值「OFF」→ 执行
```

### 把 PTT 键切换为「Mute」模式
装有 Bluetooth 模块时，PTT 键（按下音量旋钮）激活电话语音控制。  
如果不需要电话语音控制，可轻松把 PTT 键切换为「Mute」工作模式。  

!!! note "说明"
    仅对带原厂 Bluetooth 模块的车主有用。
  
``` yaml
单元 16 → 编码 → 长编码:
字节 1 – 位 3: 停用
```

### 开启泊车雷达时降低车机音量
泊车雷达开启时若车机音量过大，可降低所播放音乐的音量，使泊车雷达自身的提示音更突出。

!!! note "说明"
    车机本身的设置中也需激活此功能。
``` yaml
单元 10 → 适配: 
通道「IDE01169」: 激活
```
