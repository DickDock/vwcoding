# 动力

### 关闭挂车稳定功能 
许多人注意到关闭挂车稳定功能后会有一些效果：油耗略降、滑行距离增加、发动机制动效果减弱。 
``` yaml
单元 3 → 适配: 
通道 56 → 输入「0」→ 保存 
```

### 开启电子差速锁 (XDS) 
XDS (Electronic Differential Lock) 通过扩展的差速锁功能改善过弯动态（也称舒适驾驶）。 
``` yaml
单元 3 → 适配: 
通道 67 → 输入「1」→ 保存。只有两个值：「0」– 关闭，「1」– 激活。 
```
 
### 上/下坡驻坡辅助 (HHA) 
HHA (Hill Hold Assistant) 在上坡或下坡时保持车辆不动，防止其自行溜车，直到驾驶员踩下油门。  
HHA 等级：0 – 标准，1 – 快速（从低转速），2 – 长时（从高转速）。 
``` yaml
单元 3 → 编码 → 长编码:
字节 16 – 位 0: 激活
→ 应用
---
单元 3 → 适配: 
通道 58 → 输入所需等级值 
→ 保存 
```
 
### 紧急制动辅助设定 (BAS) 
BAS (Brake Assist System) 在危急情况下帮助驾驶员在紧急制动最初瞬间实现最大制动踏板力。  
BAS 等级：0 – 中，1 – 低，2 – 高（默认「中」） 
``` yaml
单元 3 → 适配: 
通道 09 → 输入所需等级值 → 保存 
```
 
### 制动盘清洁系统 (BDW) 
BDW (Brake Disc Wiper) – 是制动盘烘干和清洁功能。  
该功能周期性地进行不完全的短暂轻压，实质上只是让刹车片轻触制动盘（压力不超过 2 bar），从而去除湿路面在盘上形成的水膜，并周期性地清除盘上积累的污垢和油脂层。  
该功能在车速不低于 70 km/h、且雨刮已开启并持续工作不少于 5 秒时启动。  
雨刮单次开启时 – 功能触发一次。该功能在工作时驾驶员完全察觉不到。  
``` yaml
单元 3 → 适配: 
通道 55 → 输入所需值 → 测试 → 保存
```
有四个值：「0」– 关闭，「1」– Stage 1，「2」– Stage 2，「3」– Stage 3。 
 
### 扭矩补偿系统 (TSC) 
前驱车急加速时会略微向右跑偏。TSC (Torque Steer Compensation) 消除这种跑偏。 
``` yaml
单元 44 → 封闭区域 → 输入「26485」→ 适配: 
通道 05 → 输入「1」→ 保存 
```
只有两个值：「0」– 关闭，「1」– 激活。 

### 转向辅助设定 (DSR) 
Driving Steering Recommendation (DSR) 在困难条件下（例如路面有很深车辙时）辅助转向。 
``` yaml
单元 44 → 封闭区域 → 输入「51514」→ 适配: 
通道 03 → 输入「1」→ 保存 
```
只有两个值：「0」– 关闭，「1」– 激活，不过提示中是相反的。
