# 仪表盘

### 点火时的指针自检 
点火开启时，转速表和车速表指针从最小位置走到最大位置。  
没有任何实用功能，但看起来很炫！  
``` yaml
单元 17 → 适配: 
通道「indicator_celebration」→ 选择「active」→ 执行 
```
 
### 以升为单位显示油箱可加注余量 
油箱剩余空间 – 可加注升数的近似值。刻度步长 – 5 升。  
油箱加满时显示「---」。读数为近似值，可能有 ±10 л 误差。
``` yaml
单元 17 → 适配: 
通道「Volume to be replenished」→ 设为值「yes」→ 执行 
```
 
### 关闭安全带未系的声音提示和指示 
某些车辆默认开启了驾驶员和前排乘客的安全带监控。  
如果有人喜欢不系安全带驾驶，可关闭此功能。 
``` yaml
单元 17 → 编码 → 长编码:
字节 1 – 位 1: 停用 
```
 
### 校正车速表读数 
按安全规范，车速表读数相对实际车速略偏高。  
不同车辆设有不同的读数偏高等级。  
来自 Škoda Octavia (01) 的脉冲数在 100 km/h 时读数偏高 ≈ 8 km/h，来自 Škoda Fabia (02) ≈ 15 km/h。  
在 215/65 R16 规格轮胎上，采用 VW Golf (07) 的脉冲数时读数最准确。  
``` yaml
单元 17 → 编码 → 长编码:
字节 0 – 位 0-3 → 07 Distance Impulse Number 7 
```
 
### 校正燃油液位读数 
油箱燃油液位传感器存在误差，导致读数不准可达 ±10 л。 
``` yaml
单元 17 → 适配: 
通道 "Display correction of fuel gauge" → 选取值（-10 到 10）→ 执行 
```

要校正读数，需准确知道油箱里有多少汽油，或等到油灯亮起。  
油箱容量 – 64 л，每格 – 4 л。低油位报警灯在油箱剩 7 л 时亮起。
