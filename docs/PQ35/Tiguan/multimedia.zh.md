# 多媒体

### 更改 RCD-310 车机的音效方案设定 
对预设均衡器进行更优的设定。音质接近 Škoda Superb。 
``` yaml
单元 56 → 编码 → 长编码:
字节 5 – 位 0-7 → 选择 04: Škoda Superb (3Т) 
```
 
### 更改原厂车机的自主工作时长 
钥匙拔出点火锁后，车机默认工作 60 分钟。这个时间可以延长。  
无需担心蓄电池状态 – 蓄电池负载监控会在电量临界时切断用电设备（包括车机）。  
自动关闭值以分钟为单位输入。可把 60 改为不超过 99 的数字，即对应 99 分钟。 
``` yaml
单元 56 → 适配: 
通道 3 → 输入所需值 → 执行 
```
 
### 在 Swing (RCD-310) 上显示车速和转向 
所谓的「驾校」功能可在车机屏幕上显示当前车速和转向灯开启状态。 
``` yaml
单元 56 → 适配: 
通道 2 → 输入值「1」→ 保存 
```
发动机运转时关闭车机，按住 MENU 键再开机。3 秒后车速显示出现。 
 
### Swing / Bolero / Columbus 维修菜单 
从车机维修菜单可获取大量信息并进行一些设定。 
``` yaml
单元 56 → 适配: 
通道 50 → 输入值「1」→ 保存 
```
在车机开机状态下按住 MENU 键约 10 秒，会出现维修菜单。在其中可检查 FM 收音质量。
