# 照明

###  光照传感器灵敏度 
可以最优地设定光照传感器灵敏度。 
``` yaml title="登录密码: 31347"
单元 09 → 编码 → 子块 RLS → 长编码:
字节 1 – 位 0-7 → 选择所需值, 或手动指定。 
```
值越低，大灯在越暗的时候才点亮。  
推荐范围：…0707, 0808, 0909, 0A10, 0B11, 0C12, 0D13, 0E14, 0F15, 1016, 1117, 1218, 1319, 1420, 1521, 1622, 1723…  

### 紧急制动时开启双闪 
在急刹（紧急）制动时触发，表现为刹车灯或转向灯（双闪）快速闪烁。  
有 2 种工作方式：A – 通过刹车灯，B – 通过转向灯
``` yaml title="登录密码: 31347"
单元 09 → 编码 → 长编码:
(A) 字节 16 – 位 1: 停用
(B) 字节 16 – 位 2: 激活 
```

!!! warning ""
    同时激活两种方式时，紧急双闪将无法工作！ 

### 开门时点亮脚部照明 
如果通过 MFA+ 把脚部照明亮度设得较低，则开门时照明会变亮。 
``` yaml title="登录密码: 31347"
单元 09 → 适配: 
通道 27 → 输入 0 到 200 的值 (0 – 100%) → 保存 
```

### Highway 模式 
此功能在以超过 140 km/h 长时间行驶时，自动开启近光和示廓灯，即所谓的欧洲 Highway 模式。  
以低于 140 km/h 行驶超过 2 分钟时，该模式自动关闭。 
``` yaml title="登录密码: 31347"
单元 09 → 编码 → 子块 RLS → 长编码:
字节 0 – 位 0: 激活 
```

### 拉手刹时关闭 DRL 
车辆挂着手刹停着时 – DRL 关闭，一旦松开「手刹」– DRL 点亮。 
``` yaml title="登录密码: 31347"
单元 09 → 编码 → 长编码:
字节 11 – 位 0: 激活
```
如果第 0 位被隐藏，可把 16 进制值 C0 改为 C1。 

### 舒适转向灯 
舒适转向灯（即高速公路模式）在轻触转向灯拨杆（不把拨杆固定在「开启」位置）时，让转向灯闪烁一定次数。 
``` yaml title="登录密码: 31347"
单元 09 → 适配: 
通道 31 → 输入闪烁次数值（1 到 5）→ 保存 
```

### 开后备厢门时的车内照明 
默认情况下，开后备厢门时车内会亮灯（前提是车内照明开关置于「开门亮灯」位置）。  
开后备厢门时，后备厢照明灯点亮。 
``` yaml title="登录密码: 31347"

字节 20 – 位 1: 停用 
```

### 激活后备厢盖上的刹车灯 
此编码仅在带卤素大灯（NO XENON！）的改款前和改款后车型上验证过。 
``` yaml title="登录密码: 31347"
单元 09 → 编码 → 长编码:
字节 18: 把值 AA 改为 2F（千万别用 C0！） 
```

Ведомые функции → Škoda V19.47 → Octavia 2004 → 2011 (B) → Седан → CDAB 1,8 л TFSI / 112 кВт → Центральный коммутационный блок → J519 – Конфигурация освещения, параметрирование → 地址 3088 → 把值 21 改为 29。 

### 通过前雾灯激活带亮度调节的 DRL 
仅适用于没有原厂氙气和大灯内 LED 模块的人。激活前雾灯随近光一起： 
``` yaml title="登录密码: 31347"
单元 09 → 编码 → 长编码:
字节 14 – 位 1: 激活 
```
要关闭近光： 
``` yaml title="登录密码: 31347"
单元 09 → 编码 → 长编码:
字节 15 – 位 6: 停用 
```
DRL 开启时降低前雾灯亮度（55 W 的 60% 亮度，即 33 W）： 
``` yaml title="登录密码: 31347"
单元 09 → 编码 → 长编码:
字节 24 – 位 0-6 → 60 
```

### Coming Home 功能 
Coming Home 功能在锁车后让近光（或前雾灯）和示廓灯保持点亮一段时间，仿佛为你照亮回家的路。 

!!! note "2 种工作方式"
    A – 通过近光；  
    B – 通过前雾灯。  
!!! note "2 种触发方式"
    C – 开驾驶员车门后；  
    D – 熄火后。  
!!! note "2 种控制方式"
    E – 自动（由光照传感器激活）；  
    F – 手动（熄火后需「晃」一下远光）。
  
``` yaml title="登录密码: 31347"
单元 09 → 编码 → 长编码:
A) 字节 17 – 位 5: 激活 
B) 字节 17 – 位 3: 激活 
C) 字节 12 – 位 0: 激活 
D) 字节 12 – 位 0: 激活  
E) 字节 12 – 位 2: 激活 
F) 字节 12 – 位 2: 激活  
```
功能工作时长
``` yaml title="登录密码: 31347"
单元 09 → 适配: 
通道 23 → 输入工作时长值（秒，10 到 60，步长 10）→ 保存 
```

### 用前雾灯照亮转弯 (Corner) 
Corner 功能仅在 40 km/h 以下且与近光配合时工作。  
有 2 种方式：A – 根据方向盘传感器位置开启；B – 根据方向盘传感器位置和转向灯开启状态共同开启。 
``` yaml title="登录密码: 31347"
单元 09 → 编码 → 长编码:
字节 14 – 位 0: 激活 
字节 14 – 位 7: 激活 
A) 字节 17 – 位 0: 停用 
B) 字节 17 – 位 0: 激活 
```
