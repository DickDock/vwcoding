# MIB Toolbox
此工具箱旨在让你更深入地访问文件系统、修改系统皮肤与提示音、移除部分系统限制（例如导航 SD 卡的 CID 锁）、访问并更改 5F 单元的部分编码与适配、启用对文件系统的 FTP 访问等。

## MIB STD2 Toolbox
!!! warning ""
    此版本工具箱仅兼容 MIB STD2 PQ/ZR Technisat/Preh 机型，**不兼容** MIB2 STD Delphi 与 MIB2 HIGH Harman 机型！
