# FPA Editor

用于网关 **3Q0907530Q** 及更新型号的驾驶模式（Driving Profiles）参数集编辑器。支持二进制文件（4352 字节）以及 VCP 或 ODIS 格式的 XML。

[在 mqbtools.nl 上打开 FPA Editor](https://mqbtools.nl/fpa/){ target=_blank rel=noopener }

!!! tip
    作者建议使用桌面端或平板。在手机上更适合[在新标签页中打开该工具](https://mqbtools.nl/fpa/){ target=_blank }。

!!! note
    第三方服务，与 vwcoding.ru 无关。关于 FPA 刷写的更多内容 —— 见[驾驶模式配置 (FPA)](../MQB/fpaCoding.md)章节。
