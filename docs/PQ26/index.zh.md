# PQ26 平台编码

!!! note "重要信息"
    1. 这些编码大多在 Škoda Rapid 16、17 车型年上验证过。  
    2. 如果某项被标注为「需要核实」，表示已知的成功应用不超过一次。  
    3. 适用于 19 车型年及更早车辆的编码会带有「适用于 19 车型年」标注。  
    4. 如果某编码仅适用于 19 车型年车辆，会在设置名称中注明。
