# 辅助摄像头固件

为使辅助摄像头完整工作，需要正确的固件及其参数集。  
摄像头有 3 种类型：  

* 5Q0... - MFK 1.0 (Lane Assist, Sign Assist, DLA)  
* 3Q0/3QD... - MFK 2.0 (Lane Assist, Sign Assist, Traffic Jam Assist, DLA)  
* 2Q0... - MFK 3.0 (Lane Assist, Sign Assist, Traffic Jam Assist, Pedestrian Assist, DLA)  

适用于 L 和 H 固件、带延长预警时间 (60s) 的各车型参数集可在 [https://mibsolution.one](https://mibsolution.one/#/1/9/Datasets/MQB/A5/60+%20Mod) 找到

## MQB-Evo —— 摄像头 A5 (5WA 980 653)

**MQB-Evo** 平台使用 **5WA 980 653** 摄像头（非 3Q0/2Q0）。固件和参数集未经核实**不兼容**经典 MQB。

| 硬件版本          | SW 范围（示例）       | 功能                                              | SFD       |
|-------------------|----------------------|---------------------------------------------------|-----------|
| 5WA980653A        | 3129 – 3303          | Lane Assist, Sign Assist, Travel Assist（基础）   | 是        |
| 5WA980653D        | **3403+**            | + **Auto Lane Change**（带 Side Assist, PLA）     | 是        |
| 5WA980653         | 3405+                | TA 2024+，增强版 TA（取决于 PR/市场）             | 是 / SFD2 |

!!! warning ""
    - **Auto Lane Change** —— 见 [Travel Assist MQB-Evo](../MQB-Evo/travelAssist.md#auto-lane-change-opcionalno) 与[附加辅助](../MQB-Evo/driverAssistExtras.md)。  
    - Evo 的参数集请在 ODIS 中或 [mibsolution.one](https://mibsolution.one)（MEB/MQB Evo → A5 一节）按 **VIN / HW / SW** 选择。  
    - Evo 上 A5 摄像头的刷写只能通过 **ODIS online** + SFD；参数集错误 = 在恢复备份前失去辅助功能。

!!! tip "2Q0 (MFK 3.0) —— Sign Assist 与 SWaP"
    在 **2Q0980653** 摄像头上启用 **Sign Assist (VZE)** 需要 SWaP 流程 —— 见 [2Q0 摄像头 SWaP](2Q0_swap.md)。TJA —— 用下表中的参数集。
